# Install or upgrade terraform cli

## One-liner

```bash
wget https://raw.githubusercontent.com/mateothegreat/terraform-cli-installer/main/tf-install && \
chmod +x tf-install && \
./tf-install
```

## Running

```bash
./tf-upgrade [optional output dir like /usr/local/bin]
```
